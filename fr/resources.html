Les ressources pour Rails

Cette Référence Ruby on Rails Débutant ne se substitue pas à un tutoriel ou d'autres sources d'informations. Son but est de venir en complément des ressources disponibles et de vous aider. Cette page a ainsi pour objectif de partager des guides plus complets sur Ruby on Rails. Les ressources qui vont suivre, pour la plupart gratuites, ont un point en commun: Elles sont adaptées aux débutants!

Lecteurs francophones, la grande majorité des liens présentés dans cette page sont exclusivement en anglais. Si vous n'êtes pas très à l'aise avec la langue de Shakespeare, le site jeveuxapprendreruby.fr liste des ressources avec un accent sur le contenu français. Pour les bilingues, lisez ci-dessous.

Les ressources recommandées

Voici une liste de ressources recommandées pour apprendre Ruby on Rails. Si vous connaissez un tutoriel qui devrait se trouver dans cette liste, merci de nous le dire. Les idées et remarques sont toujours les bienvenues!

  • Le livre tutoriel Ruby on Rails de Michael Hartl - L'un des meilleurs tutoriels Ruby on Rails disponibles - c'est presque incroyable qu'il soit gratuit. Michael Hartl vous guide intégralement dans la procédure pour installer Ruby on Rails, vous en apprend les bases, puis développe, avec vous, une application ressemblant à Twitter. Le tout via un développement piloté par les tests (Test Driven Development). C'est également la façon dont j'ai appris Ruby on Rails ;-).
  • RailsCasts - Bien qu'il n'ait pas produit de nouveaux épisodes depuis mi-2013, Ryan Bates fait un travail incroyable pour expliquer succinctement les fonctionnalités de Ruby on Rails ou présenter des gemmes via d'incroyables vidéos d'écrans. Souvent, il vous guide au long d'un itinéraire étape par étape pour accomplir une tâche spécifique. Il est le moyen pour de nombreux développeurs de découvrir des gemmes ou fonctionnalités vraiment cools.
  • Les tutoriels JumpstartLab - JumpstartLab propose des formations et ateliers Ruby on Rails. Ils partagent l'intégralité de ce contenu gratuitement sur internet ! Ils ont aussi une liste avec des sujets plus spécifiques. Génial!
  • Ruby on Rails Guides - Ici vous pouvez trouver les guides officiels couvrant Rails dans son ensemble, allant de sujets pour débutants à d'autres plus avancés.
  • Codecademy - Un site intéractif remarquable, en français, pour apprendre Javascript, les bases des pages internet (HTML + CSS), le language Python et bien évidemment Ruby. Ce site est réellement intéractif, vous pouvez ainsi aller sans attendre aux exercices, directement dans votre navigateur. De plus, vous pouvez suivre votre parcours et gagner des badges. J'adore les badges, pas vous?

Un parcours organisé

Ce parcours pour débuter dans Ruby on Rails vous est proposé par Blair Anderson et est un ensemble ordonné de ressources adaptées pour entamer votre apprentissage.

  1. Nitrous Online Development Environment

    Si vous rencontrez des difficultés avec votre environnement Ruby, allez voir Nitrous, cela vous permet de créer des applications Rails dans votre navigateur.

  2. Tutoriel JumpStartLab pour créer sa première application Rails

    Ce tutoriel ne requiert aucune connaissance préalable de Rails et explique grossièrement son fonctionnement interne. Ce tutoriel démarre avec git/github.

  3. Regarder quelques Railscasts

    Les Railcasts sont des vidéos d'écrans très populaires pour apprendre Ruby on Rails. La plupart du temps, si vous voulez apprendre quelque chose concernant Rails, vous pourrez trouver un railscast qui vous montrera comment procéder.

  4. Apprendre Ruby

    Après avoir effectué cette introduction de base pour Rails, nous conseillons de passer plus de temps spécifiquement sur Ruby. Codecademy est parfait pour vous apprendre les leçons de base, et ramener ces connaissances vers Rails.

  5. Le tutoriel Ruby on Rails de Michael Hartl

    Le livre de Michael Hartl est un grand plongeon pour les débutants et vous apportera une grande connaissance de Rails.

  6. Capybara Railscast et Le tutoriel Rspec et BDD

    Une fois Rails compris, il est important de savoir écrire de solides applications à l'aide de tests. Nous recommandons fortement d'apprendre comment tester votre code Ruby avec Rspec (ou Minitest) et Capybara.

  • Le site de ressources de PragTob - Une liste de ressources proposant à la fois des ressources Ruby on Rails pour débutants et apprentis avancés ainsi que d'autres types de ressources.
  • Les ressources de Rails Girls - Une liste de ressources provenant du génial Rails Girls.
  • Liste de tutoriels Rails par RailsApps - Ce qui est vraiment cool avec cette liste c'est que vous pouvez la filtrer par critère: niveau de compétence, format, structure et bien plus!
