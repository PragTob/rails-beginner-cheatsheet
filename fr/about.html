À propos de

Bienvenue ! Vous commencez l'apprentissage de Ruby on Rails ? Fantastique ! Cette page est pour vous. Elle résume certaines des informations les plus importantes à propos de Ruby et de Rails, pour référence. Ce sont les choses qu'on oublie facilement, comme comment démarrer le serveur ou à quoi sert ce dossier de configuration déjà ? De plus, il vise spécialement les débutants, ce qui signifie un peu moins de concepts mais davantages d'explications, par rapport à d'autres fiches de référence (Cheat Sheets). Nous pensons que cela peut aider tous les débutants en Rails, que vous ayez des années d'expérience dans d'autres langages et plateformes, ou que vous soyez totalement débutant en développement. Cela dit, c'est ce dernier groupe que nous ciblons en priorité :)

Notez qu'il s'agit d'une première version, sur un logiciel en constante évolution. Il peut contenir des informations incorrectes ou erronnées, ou se référer à une version de Rails qui n'est pas strictement la dernière. La précision n'est pas notre souci premier, c'est la simplicité. Nous voulons aider les gens à débuter avec Ruby on Rails, pas les perdre dans les méandres de chaque cas extrême qu'un expert pourrait imaginer.

Cette fiche n'a pas pour but d'être un guide complet sur Rails. Si c'est ce que vous recherchez, il existe plusieurs compilations de ressources pour apprendre Rails (et autres), souvent en anglais (par exemple le blog de l'auteur) mais parfois aussi en français (comme jeveuxapprendreruby.fr).

Problèmes / Contributions

Il y a peut-être des erreurs, coquilles ou choses à préciser. S'il manque quelque chose d'important, si une explication n'est pas claire, si le design est à améliorer... Si c'est le cas, contactez nous ! Vous pouvez remonter les erreurs sur GitHub (issues), où vous aurez aussi une vision des problèmes connus à l'heure actuelle. Si vous n'avez pas de compte GitHub, envoyez un mail aux auteurs PragTob ou Sylvain Abélard. Nous ferons de notre mieux, mais si vous voulez et pouvez, une Pull-Request sur le GitHub du projet sera appréciée :)
